# 墨脉 (Ink-Pulse)：一支跨越 8 年的智能交互笔
> **从 2017 年的小学征文《自动电子笔》，到 2025 年的跨学科原型探索**

## 📜 缘起：儿时的发明家梦 (The Origin)
本项目起源于我小学时期的一篇获奖征文。在那时，我幻想有一支能“教人写字、纠正姿势”的笔。8 年后的今天，作为浙大的学生，我尝试用目前自学的技术去回应这个童年的梦想。

---

## 💡 设计思路：从感性构思到理科逻辑
[cite_start]我尝试将生物科学中的“反馈机制”与工业设计中的“人机交互”结合，探索如何用硬核技术找回书写的“温度” [cite: 1, 23]。

### 1. 感知与对比 (Sensing & Comparison)
- [cite_start]**物理感知**：利用压力传感器实时捕捉书写的精微数据 [cite: 37, 43]。
- [cite_start]**算法逻辑**：将采集到的压力、坐标数据与名家碑帖的风格矩阵进行比对 [cite: 13, 47]。

### 2. 交互与反馈 (Interaction & Feedback)
- [cite_start]**触觉引导**：通过微型电机产生物理反馈（如模拟纸笔摩擦感），引导用户运笔 [cite: 6, 48]。
- [cite_start]**形态自适应**：探索利用 4D 打印材料，根据用户握力实时调整出最优人体工学握姿 [cite: 9, 38]。
